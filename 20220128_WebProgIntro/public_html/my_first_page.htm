1 Демонстрация возможностей HTML

1.1 Демонстрация списков

Как распределяются обязанности между разработчиками веб-приложения?
  • ДИЗАЙНЕР - разрабатывает внешний вид (пользовательский интерфейс) веб-сайта в графическом редакторе (например Photoshop)
  • ВЕРСТАЛЬЩИК - реализует на HTML и CSS графический макет, созданный дизайнером.
  • ПРОГРАММИСТ - реализует логику веб-приложения с использованием алгоритмических языков программирования (PHP, JavaScript), обеспечивает работу с СУБД.
  • ПОИСКОВЫЙ ОПТИМИЗАТОР (SEO) - отвечает за продвижение сайта в поисковых системах.
